Types of programming: terms and basic languages

Types of programming

There are 7 basic types (categories) of programming:
1. Aspect-oriented languages
2. Structural
3. Procedural
4. Logical
5. Object-oriented
6. Functional
7. Multi-paradigm
Aspect-oriented programming A programming paradigm based on the idea of separating functionality to improve the division of a program into modules.
The most popular languages are: AspectC++; AspectJ; AspectLua
Structural programming A programming paradigm based on the representation of a program in the form of a hierarchical block structure.
The most popular languages are: Basic; Pascal; QBASIC
Procedural programming Programming in an imperative language, in which sequentially executed statements can be assembled into subroutines, that is, larger integral units of code, using the mechanisms of the language itself.
The most popular languages are: Pascal; PHP; Lua
Logic programming A programming paradigm based on mathematical logic — programs in it are set in the form of logical statements and rules of inference.
The most popular languages are: Mercury; Prolog; Curry
Object-oriented programming A programming methodology based on the representation of a program as a collection of interacting objects, each of which is an instance of a certain class, and the classes form an inheritance hierarchy.
The most popular languages are: C++; C#; Java
Functional programming A programming paradigm in which the calculation process is interpreted as calculating the values of functions in the mathematical understanding of the latter (as opposed to functions as subroutines in procedural programming).
The most popular languages are: F#; Cat; Clean
A multi-paradigm programming language A programming language that was developed specifically as a multi-paradigm programming tool, that is, the expressive capabilities of which were originally supposed to be inherited from several, most often unrelated languages of various paradigms.
The most popular languages are: Python; C++; JavaScript
